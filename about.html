About this project

Citation & H-Index Dashboard
คณะแพทยศาสตร์

ระบบแสดงผลงานวิชาการของคณาจารย์ สร้างขึ้นจากข้อมูล Scopus เพื่อติดตามผลกระทบงานวิจัย (research impact) ผ่าน citations, h-index และการจัดอันดับวารสาร

ที่มาของโปรเจกต์

โปรเจกต์นี้เกิดขึ้นจากความต้องการติดตามผลงานวิชาการของอาจารย์ประจำคณะแพทยศาสตร์ มหาวิทยาลัยนเรศวร ในรูปแบบที่เข้าถึงได้ง่ายและสามารถค้นหา จัดเรียง และวิเคราะห์ข้อมูลได้แบบ interactive แทนการดูข้อมูลจาก spreadsheet

ข้อมูลต้นทางรวบรวมจากฐานข้อมูล Scopus ครอบคลุมบทความที่ตีพิมพ์ในช่วงปี 2022–2026 รวม 264 บทความ จากอาจารย์ประจำ 198 คน โดยนำมาประมวลผลและแสดงผลผ่านเว็บแอปพลิเคชันที่สร้างด้วย AI

AI
สร้างโดย Claude AI (claude-sonnet-4-6) ทั้งการวิเคราะห์ข้อมูล (Python/Pandas), การออกแบบ UI และการเขียนโค้ด HTML/CSS/JavaScript ล้วนสร้างขึ้นผ่าน Claude AI ใน session เดียว โดยใช้เวลาไม่ถึง 1 ชั่วโมง

แหล่งข้อมูล

📂 ไฟล์ต้นทาง template_citation.xlsx — ข้อมูล Scopus export จัดทำโดยฝ่ายวิจัย คณะแพทยศาสตร์ มหาวิทยาลัยนเรศวร
🗄 ฐานข้อมูล Scopus (Elsevier) — ฐานข้อมูลงานวิจัยนานาชาติ ครอบคลุมวารสาร peer-reviewed กว่า 25,000 รายการ ใช้สำหรับอ้างอิง citations, h-index และ quartile ranking
📊 Looker Studio Dashboard ข้อมูลเชื่อมโยงกับ Google Looker Studio สำหรับการวิเคราะห์เชิงลึกเพิ่มเติม

Timeline การพัฒนา

Scopus Database · ต่อเนื่อง
รวบรวมข้อมูลผลงานวิชาการ
ฝ่ายวิจัยคณะแพทยศาสตร์รวบรวม citations, h-index และ quartile ranking จากฐานข้อมูล Scopus สำหรับอาจารย์ 198 คน ครอบคลุมบทความปี 2022–2026 รวม 264 บทความ
กรกฎาคม 2025 · Phase 1
จัดทำ Excel Template
สร้างไฟล์ template_citation.xlsx รวมข้อมูล h-index, citations รายปี (2022–2026), quartile (Q1–Q4) และสถิติรวมของคณะ พร้อม link ไปยัง Looker Studio
กรกฎาคม 2025 · Phase 2
วิเคราะห์ข้อมูลด้วย Claude AI
อัปโหลดไฟล์ Excel เข้า Claude AI — ระบบ AI อ่าน, แยกโครงสร้างข้อมูล และสรุปสถิติสำคัญ ได้แก่ การกระจายตัวตาม quartile, บทความที่ถูกอ้างอิงสูงสุด และแนวโน้ม citations ตามปี โดยใช้ Python/Pandas
กรกฎาคม 2025 · Phase 3
สร้าง Interactive HTML Dashboard
Claude AI สร้างหน้า index.html แบบ standalone พร้อม embedded data 264 บทความ มีฟีเจอร์ค้นหา, กรองตามปี/quartile/h-index, จัดเรียงทุกคอลัมน์ และแถบสรุป citations พร้อม visual bar
กรกฎาคม 2025 · Phase 4
สร้างหน้า About & Navigation
เพิ่มหน้า about.html อธิบายที่มา เทคโนโลยี timeline และผู้จัดทำ พร้อม navigation bar เชื่อมสองหน้าเข้าหากัน
กรกฎาคม 2025 · Phase 5 — ปัจจุบัน
เผยแพร่ผ่าน GitHub + Cloudflare
อัปโหลดไฟล์ index.html และ about.html ไปยัง GitHub repository เชื่อมต่อกับ Cloudflare Pages เพื่อ CDN ทั่วโลกและ custom domain พร้อมอัปเดตข้อมูลอัตโนมัติเมื่อ push code
อนาคต · Planned
อัปเดตข้อมูลประจำปี
วางแผนอัปเดตข้อมูลจาก Scopus ทุกปีการศึกษา พร้อมเพิ่มกราฟแสดงแนวโน้ม และรองรับการค้นหาตามชื่อผู้แต่งแบบ autocomplete

วิธีเผยแพร่

เว็บไซต์นี้เป็น static HTML ทั้งหมด ไม่มี backend หรือ database — ข้อมูลทั้ง 264 บทความถูก embed ไว้ในไฟล์ HTML โดยตรง ทำให้ deploy และ maintain ได้ง่ายมาก

ใช้ GitHub เป็น source code repository และ Cloudflare Pages เป็น hosting ที่มี CDN กว่า 300 จุดทั่วโลก ทำให้โหลดเร็วจากทุกประเทศ — และทั้งหมดนี้ ฟรี
